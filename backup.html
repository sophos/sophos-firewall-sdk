  • Password (str, optional) – FTP Server password

  • FtpPath (str, optional) – FTP Server path

  • EmailAddress (str) – Email address

  • BackupFrequency (str) – Never/Daily/Weekly/Monthly

  • Day (str) – Day

  • Minute (str) – Minute
